# PlanMath



Este software es un diseñador visual y ejecutor de procesos matemáticos, es decir, operaciones aritméticas elementales en forma de flujo o proceso.



Se tiene un diseñador visual y un motor de ejecución.

Está la motivación de agregarle módulos de ejecución compartida, por ejemplo, servicios a la nube, creación de archivos de diseño de flujos y operaciones para compartir y comercializar en futuras tiendas online.
